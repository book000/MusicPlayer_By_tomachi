# MusicPlayer_By_tomachi  
ネットワーク上にある音楽ファイルを読み込み再生してTwitterにツイートするプログラム  

## 使用ライブラリ  
以下ライブラリを使用しています。  
- getID3 <https://github.com/JamesHeinrich/getID3> 
- TwitterOauth <https://github.com/abraham/twitteroauth>
